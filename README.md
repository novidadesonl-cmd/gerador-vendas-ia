# Gerador de Vendas IA

MVP de uma ferramenta que gera conteúdos de venda a partir de um formulário simples.

## Objetivo

Permitir que pequenos negócios, afiliados, infoprodutores e prestadores de serviço gerem conteúdos prontos para vender no Instagram, Reels, WhatsApp, anúncios e páginas de venda.

## Primeira versão

A primeira versão tem:

- Página inicial com posicionamento comercial da ferramenta
- Formulário de amostra grátis
- Resultado simulado com blocos copiáveis
- Página de planos com três ofertas e links placeholder da Kiwify
- Design responsivo com aparência moderna de ferramenta de IA

## Stack

- React
- TypeScript
- Vite
- CSS simples

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
